# Macronutrient Calculator

## Overview

Macronutrient Calculator adalah aplikasi web sederhana yang digunakan untuk menghitung kebutuhan makronutrisi harian seseorang berdasarkan data tubuh dan tujuan nutrisi.

Aplikasi ini membantu pengguna memperkirakan kebutuhan kalori harian serta pembagian makronutrisi utama yaitu protein, karbohidrat, dan lemak. Perhitungan dilakukan berdasarkan data dasar pengguna seperti berat badan, tinggi badan, usia, jenis kelamin, dan tingkat aktivitas.

Website ini dapat digunakan sebagai alat bantu sederhana bagi individu yang ingin mengatur pola makan, menurunkan berat badan, meningkatkan massa otot, atau menjaga pola makan yang lebih terstruktur.

---

# Features

## Daily Calorie Calculation

Menghitung estimasi kebutuhan kalori harian pengguna berdasarkan data tubuh dan tingkat aktivitas.

## Macronutrient Distribution

Menghitung pembagian makronutrisi utama dari total kalori harian:

* Protein
* Karbohidrat
* Lemak

Hasil perhitungan ditampilkan dalam bentuk gram per hari.

## User Input Form

Pengguna dapat memasukkan beberapa informasi dasar seperti:

* Berat badan
* Tinggi badan
* Usia
* Jenis kelamin
* Tingkat aktivitas

Data tersebut digunakan untuk menghitung kebutuhan energi dan makronutrisi.

## Instant Result

Setelah data dimasukkan, sistem akan langsung menampilkan hasil perhitungan kebutuhan kalori serta distribusi makronutrisi.

---

# Calculation Logic

Perhitungan dilakukan melalui beberapa tahap:

1. Menghitung **Basal Metabolic Rate (BMR)** berdasarkan data tubuh pengguna
2. Menyesuaikan nilai BMR dengan **tingkat aktivitas harian** untuk mendapatkan **Total Daily Energy Expenditure (TDEE)**
3. Membagi total kalori menjadi makronutrisi utama:

   * Protein
   * Karbohidrat
   * Lemak

Setiap makronutrisi kemudian dikonversi dari kalori menjadi gram untuk memudahkan pengguna dalam mengatur konsumsi makanan.

# How to Use

1. Buka halaman kalkulator
2. Masukkan data tubuh pengguna
3. Pilih tingkat aktivitas
4. Klik tombol **Calculate**
5. Sistem akan menampilkan estimasi kebutuhan kalori dan makronutrisi harian
